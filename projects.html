Shamir Secret Sharing and Distributed Key Generation (DKG)

February 2025 - Present

Developed a cryptographic system in Python, implementing Shamir’s Secret Sharing (SSS) and Distributed Key Generation (DKG) protocols. This project focuses on securely splitting, distributing, and reconstructing sensitive data using threshold cryptography, ensuring no single party can compromise the secret.

Key technical aspects include:

  • Implementation of Shamir’s Secret Sharing to divide a secret into multiple shares, requiring a predefined threshold of shares for reconstruction.
  • Integration of Distributed Key Generation (DKG) to enable secure, decentralized key management among multiple parties without a trusted dealer.
  • Use of modular arithmetic and polynomial interpolation to ensure cryptographic security and correctness.
  • Used matplotlib to demonstrate the secret sharing and reconstruction process, making complex cryptographic concepts more accessible and intuitive.

This project is still a work in progress, with ongoing efforts to enhance functionality, improve performance, and expand features. Future updates will include additional cryptographic protocols, optimizations for scalability, and further refinements to the visualization component.

This work has deepened my understanding of threshold cryptography, secure secret sharing, decentralized systems, and some fun (but tricky) math! Sadly, LinkedIn won't let me attach my GitHub traditionally, so feel free to check it out here: GitHub Repository.

Technologies Used:

  • Python
  • Shamir’s Secret Sharing
  • Distributed Key Generation (DKG)
  • Matplotlib
  • Modular Arithmetic
  • Polynomial Interpolation
